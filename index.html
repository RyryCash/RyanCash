Night

Welcome to GameNight! This application is designed to bring friends and family for board game nights. The application will allow users to create an account, create a game night, and invite friends to join. The application will also allow users to search for games and add them to their game night. We have two frameworks that we are using for this application. The first is React, which is a JavaScript library for building user interfaces. The second is FastAPI, which is a modern, fast (high-performance), web framework for building APIs with Python 3.6+ based on standard Python type hints. In GameNight we will be using a PostgreSQL database. PostgreSQL is a powerful, open source object-relational database system with over 30 years of active development that has earned it a strong reputation for reliability, feature robustness, and performance.

CarCar

"CarCar" is a comprehensive application crafted to simplify and optimize the daily operations of automobile dealerships. Utilizing Django for a robust backend and React for an interactive frontend, the platform offers comprehensive features ranging from sales tracking and customer management to efficient automobile inventory control and service scheduling. Dealerships can effortlessly monitor sales, understand customer histories, evaluate sales personnel performance, and ensure timely service delivery, all from one cohesive interface. This application not only streamlines administrative tasks but also empowers dealerships to create more personalized customer interactions, ultimately driving business growth and enhancing user experience. The premiere solution for automobile dealership management!.
